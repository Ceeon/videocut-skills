# 静音段处理

## 阈值规则

| 静音长度 | 处理 |
|---------|------|
| < 1秒 | **忽略** - 自然停顿 |
| ≥ 1秒 | 建议删除 |

## 输出格式

**按 1 秒一个格子输出，不要合并**

示例：3.2 秒静音 → 输出 4 个独立项
```
- [ ] 静音 10.0-11.0 (1.0s)
- [ ] 静音 11.0-12.0 (1.0s)
- [ ] 静音 12.0-13.0 (1.0s)
- [ ] 静音 13.0-13.2 (0.2s)
```

这样用户可以精细控制保留哪段。
